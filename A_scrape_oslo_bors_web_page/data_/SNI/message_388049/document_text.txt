CORRECTION OF FIDUCIA SHAREHOLDING

This notification serves as a correction to the number 
of shares owned by Fiducia Ltd.

On October 6, 2015, Fiducia Ltd. reported that it had 
acquired 142,000 shares in Stolt-Nielsen Limited at an 
average price of NOK 118.50 per share, after which 
Fiducia Ltd. owned 29,672,255 shares in Stolt-Nielsen 
Limited. This is incorrect. The correct number of 
shares owned by Fiducia Ltd. is 29,634,255 shares.
